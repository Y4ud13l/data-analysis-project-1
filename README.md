# Proyecto de Aprendizaje en Análisis de Datos

Este repositorio documenta un proyecto personal de aprendizaje en análisis de datos.

## Herramientas utilizadas
- MySQL
- SQL
- Python (Anaconda)
- Excel
- Power BI

## Estructura del repositorio
- sql: consultas SQL
- python: notebooks y scripts
- excel: archivos de validación
- powerbi: dashboards
- docs: documentación del proceso

El objetivo del proyecto es aprender el flujo completo de análisis de datos,
documentando decisiones, errores y resultados.
